# Virtus Barbearia

Landing page institucional desenvolvida para apresentação comercial da Virtus Barbearia, com foco em presença digital premium, divulgação da marca e conversão para agendamento via WhatsApp.

## Objetivo

Este projeto foi criado como um MVP de apresentação comercial para demonstrar como a Virtus Barbearia pode fortalecer sua presença online com uma landing page moderna, responsiva e orientada à conversão.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Funcionalidades

- Hero section com proposta de valor
- Seção de diferenciais
- Apresentação da barbearia
- Lista de serviços
- Galeria de imagens
- Localização com integração ao Google Maps e Waze
- Botão flutuante para contato via WhatsApp

## Como rodar localmente

```bash
npm install
npm run dev
```

A aplicação ficará disponível em:

```bash
http://localhost:3000
```

## Build de produção

```bash
npm run build
```

## Estrutura do projeto

```bash
app/
components/
public/galeria/
```

## Observações

- O projeto está configurado para exportação estática.
- As imagens estão otimizadas para um MVP de apresentação.
- A estrutura foi pensada para futura evolução com refinamento visual, automação de atendimento e agendamento online.
